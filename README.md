# 📚 Study Planner Web App

A responsive Study Planner Web Application developed using PHP, MySQL, HTML, CSS, and JavaScript. The application helps students organize study schedules, manage daily tasks, track progress, and improve productivity.

## 🚀 Features

* User Registration & Login
* Add, Edit, and Delete Study Tasks
* Mark Tasks as Completed
* Calendar-Based Task Management
* Dashboard Overview
* User-Specific Task Tracking
* Session-Based Authentication
* Responsive Design

## 🛠️ Tech Stack

**Frontend**

* HTML5
* CSS3
* JavaScript
* Font Awesome

**Backend**

* PHP

**Database**

* MySQL

## ⚙️ Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/study-planner.git
   ```

2. Move the project folder to the XAMPP/WAMP `htdocs` directory.

3. Start Apache and MySQL.

4. Create a MySQL database.

5. Import the `schema.sql` file.

6. Configure database credentials in `db.php`.

7. Open:

   ```
   http://localhost/Project
   ```
